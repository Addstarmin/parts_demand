# CMD-X 5分デモシナリオ

1. React画面を開き、ダッシュボードで工場を「F-02 三河精密工場」にする。
2. 対象を「製品アセンブリ」に切り替え、「PROD-A キャブレターASSY」を選ぶ。
3. 製品需要の4週間予測を見せ、製品単位の過去需要履歴から予測していると説明する。
4. 「構成部品別」に切り替え、PT-1002などの次週必要数、4週間合計、現在庫、安全在庫、不足見込みを説明する。
5. シミュレーション画面へ移動し、完成車A社、PROD-A、調整率-20%で実行する。
6. 通常予測と調整後予測の破線比較、部品別削減候補、計算時間msを説明する。
7. 安全在庫最適化画面へ移動し、現在設定、RMSE、LT、ガードレールを説明する。
8. プレビューを実行し、増加・減少・変更なし・要確認を確認する。
9. 「今すぐ最適化」を実行し、成功件数を確認する。
10. ダッシュボードへ戻り、部品予測の安全在庫根拠が更新値を参照していることを示す。
